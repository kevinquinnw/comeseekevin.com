Projects

Wildfire Predictor - (Github Repo)

The goal of this project was to obviously predict wildfires but we decided to see if we could predict the amount of wildfires in any state in the United States based off previous wildfire counts, average precipitation and the annual precipitation. To do this we used a negative binomial regression and a poisson regression. I jointly created this project with some of my peers in CSC110 in 2020. Created with Python and many other libraries.

Polaroid Generator - (Github Repo)

A web app I made that captures a photo by fetching the current video through a webcam, drawing it to a canvas and converting to a PNG. It was one of my first projects using some basic front-end languages such as Javascript, HTML, and CSS.

Sudoku Solver - (Github Repo)

Sudoku solver that uses a backtracking algorithm to solve a given problem by attempting all possible routes towards a solution. One of the best projects anybody can do when they first start learning programming is to create a sudoku solver because it's a great way to learn recursion. Created with Python.
